Amazon Alexa to Revit

Voice command Alexa to Revit

Description

An Alexa to Revit prototype uses voice activation to generate geometry in Revit. The Alexa also gives feedback such as quantities from a schedule in Revit.

The project uses Alexa Skills Developer Kit. The skill is given a name such as ‘the architect’. The payload is given as a JSON file which is parsed out using Python and the library Pandas. The Pandas writes to a CSV file that has 0 and 1 for switchboard. Dynamo is monitoring the switchboard CSV file and runs the appropriate command in Revit.
